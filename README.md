AI Book Insight System is a backend application developed using Django that allows users to store and manage book information. The system automatically generates
a concise summary and predicts the genre of each book based on its description using intelligent text-processing logic.
The application provides REST API endpoints to add new books and retrieve stored data. It also includes an admin panel for easy data management.
This project demonstrates backend development, API creation, and basic AI-driven text analysis.

Key functionalities include:
-Book data storage (title, author, description)
-Automatic summary generation
-Genre classification
-RESTful API integration
-Admin

   FEATURES
📄 Upload or paste book text
🧠 AI-powered summarization
🔑 Keyword extraction
📊 Key idea identification
💡 Smart insights from text
⚡ Fast backend processing
🌐 Simple web interface (frontend + backend)
